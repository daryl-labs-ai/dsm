# Security

## Reporting vulnerabilities

**Do not open public issues for security vulnerabilities.**

If you believe you have found a security issue in DSM, please report it privately:

- **Contact:** [security@daryllabs.ai](mailto:security@daryllabs.ai)

Include a clear description, steps to reproduce, and any impact assessment. We will acknowledge receipt and work with you to address the issue.

## Supported versions

- **main** branch: supported with security updates as needed.

Other branches are not officially supported for security fixes.

## Disclosure policy

- We will work with reporters to understand and fix validated vulnerabilities.
- We aim to release a fix and coordinate disclosure in a reasonable timeframe.
- Credit will be given to reporters when they agree.
